# Pemrograman Rust untuk Pemula

Buku ini merupakan buku pemrograman Rust yang ditujukan terutama untuk memperkenalkan bahasa pemrograman Rust serta ekosistem Rust dalam membangun berbagai macam peranti lunak. Tujuan akhir dari buku ini adalah pengetahuan dasar yang kuat tentang pemrograman menggunakan Rust serta kemampuan mendayagunakan ekosistem Go untuk membangun aplikasi. Setelah mempelajari buku ini, diharapkan pembaca bisa memahami materi Rust tingkat menengah maupun lanjut seperti ``secure programminig``, ``konkurensi``, ``pemrograman sistem``, dan lain-lain.
